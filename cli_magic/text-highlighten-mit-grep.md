# Text Highlighten mit grep \(ohne zu Filtern\) {#text_highlighten_mit_grep_ohne_zu_filtern}

```
$ grep --color -E '^|pattern1|pattern2' file name
```

Oder:

```
$ <command_here> | grep --color -E '^|pattern1|pattern2'
```
