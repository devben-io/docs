# GIT \| Subtrees

## Setup

```
git remote add <MY-SUBTREE> git@github.com:<USER>/<SUBPOJECT>.git
git subtree add --prefix=<MY-SUBTREE>/ <MY-SUBTREE> master
```

## Pull changes from sub-repo

```
git subtree pull --prefix=<MY-SUBTREE> <MY-SUBTREE> master
```
