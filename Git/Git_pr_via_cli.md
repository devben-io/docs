# GIT - open PR with CLI

## Install Tool "hub"
e.g. for ARCH Linux:
```bash
yay -S hub
```

## SSH instead of git protocol 
```bash
$ git config --global hub.protocol ssh
```

## Open PR on Github
```bash
hub pull-request -o
```
**`-o`**  open PR in Browser afterwards
