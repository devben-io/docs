# VScode Cheatsheet / Hacks

## Overview
This is a Cheatsheet for some VScode hacks

## Prerequisite
* vscode

## Cheatsheet
```
# Edit all occurences
- Select text
- [CTRL] + [SHIFT] + [L]

# Multiline Edit
- [CTRL] + [SHIFT] + [ARROW UP/DOWN]

# Multi-cursor Edit
- [ALT] + select text parts

# Change text to UPPER-/lowercase
- Select text 
- [CTRL] + [SHIFT] + [P] 
- Select / type “Terraform to Uppercase” / “Terraform to Lowercase” 

```
